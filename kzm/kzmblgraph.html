KzmBLGraph

    Date: 22.Mar.96

    sdfasdfasdfasdf

    This applet is an evolution of my previous KzmStatBar applet.
    Is a simple way to have a horizontal or vertical bar or line graph, basing on a very simple text file, that contains the label, the value and the color (RGB format) of every point of the graph.
    The bars (or line) are automatic calculated basing on the number of values and (if present) the title.

    PARAMETERS

    textcolor: (Red_Value Green_Value Blue_Value, default = 0 0 0)
    Is the color of all the texts (labels, title and line in line graph).
    backcolor: (Red_Value Green_Value Blue_Value, default = 192 192 192)
    Is the color of the background.
    backimage: (String, default = none)
    If you wanna use an image as background, define it here. Consider that the image overrides the background color and make the applet slower (The applet is show only when all image is completely loaded).
    filename: (String, default = stat.txt)
    Is the name of the file containing data. Wanna see the example ? Click data.txt.
    type: (hbar, vbar, line default = hbar)
    The type of graph: vbar = vertical bar, hbar = horizontal bar, line = line graph. Example is (obviously) vbar.
    title: (String, default = none)
    If present, is the title of the graph.
    copyright: (String)
    Ok, this is a copyright line. Please don't change it. This applet is freeware, so the only cost is to leave this copyright parameter.
