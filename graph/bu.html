How to use this applet


This applet draw three tipes of graph from the data file.
  • choose the type of the graph ( Line,Rectangle,Circle )
  • Input the number into the textfield.
    • if you chose Line or Rect for kind, input the number that represent the kind of data.)
    • or else you chose Circle for kind, input the number that represent time plot
  • click "submit" to draw the graph.
  • click "clear" to clean the display.
This is the developed version of graph drawing applet NF4. Source code is available, but is still messy. Since this is the first applet I've written.
I found one bag. Sometimes the window for choosing the kind of graph doesn't appear. In this case just try to ckick beside the text field, the window might appear then.
