The AT&T WWW Mailing List

The AT&T WWW mailing list is a discussion group for anything related to AT&T and the WWW. In order to subscribe to the AT&T WWW list you must send a message to hotair!att.infosystems.www-request with a message body of: 

subscribe www <your human name, i.e., Joe Btfsplk>

eg:

subscribe www Joe Btfsplk
This system is automated so don't bother including other text.

In order to post to the mailing list send email to att.infosystems.www@hotair.tbu.att.com, but don't use this address for administrative purposes or you will send a message to everyone on the list.

Generating GIF Images for the Web

The latest thing I am investigating is generating GIF images for web pages. The gd library written by Thomas Boutell generates gif images with a limited set of graphics functions. A patch was provide by Viet Hoang integrates this library with gnuplot to create graphs on the fly.
